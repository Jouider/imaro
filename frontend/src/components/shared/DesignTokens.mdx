## Typographie

- **Body** — `Nunito Sans` (`font-sans`)
- **Headings** — `DM Serif Display` (`font-display`) sur les H1 de page
- **Numérique** — toujours `tabular-nums` pour les montants et stats

> Jamais d'`Inter`, `Roboto` ou `Arial` — la marque s'écrit en Nunito Sans + DM Serif.

## Règles d'usage

- **Montants** — toujours `<MontantDisplay value={...} />` (locale fr-FR + `DH`, jamais MAD/€/$).
- **Statuts** — toujours `<StatutBadge statut="..." />` (i18n géré).
- **Suppression** — toujours via `<ConfirmModal>` avant la mutation.
- **Loading** — `<LoadingSkeleton variant="..." />`, jamais de spinner nu.
- **CTA orange** — `bg-[var(--accent)] text-white hover:bg-[var(--color-imaro-accent-dark)]`.
- **Pas de texte UI en dur** — toujours i18n (`useTranslation`), FR + AR (RTL via `<html dir>`).
